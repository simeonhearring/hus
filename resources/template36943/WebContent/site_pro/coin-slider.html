In order to create a Coin slider you will need:

  1. Create a <div> with coin-slider id(<div id="coin-slider">)
  2. Each slide should consist of the following coding structure:
    <a href="more.html"><img src="images/pic1.jpg"><div class="banner"> Description</div></a>
  3. You can also add a slide description using the <div class="banner"> element (<div class="banner">Description text</div>)
